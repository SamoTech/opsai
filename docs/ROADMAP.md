# OpsAI Product Roadmap

## v1.0 — MVP (Current)

- [x] GitHub Actions webhook integration
- [x] AI log analysis with Groq LLM
- [x] Root cause classification (7 categories)
- [x] Fix suggestions with code snippets
- [x] Slack notifications
- [x] Pipeline dashboard
- [x] Docker Compose deployment

## v1.1 — Integrations (Q2 2026)

- [ ] GitLab CI integration
- [ ] Jenkins integration
- [ ] Bitbucket Pipelines
- [ ] PagerDuty + OpsGenie alerts
- [ ] GitHub PR comments with fix suggestions

## v1.2 — Intelligence (Q3 2026)

- [ ] Pattern learning across runs ("This error recurs every Monday")
- [ ] Auto-fix PRs via GitHub API
- [ ] Multi-step log correlation
- [ ] Custom classification rules
- [ ] LLM provider selection (OpenAI, Anthropic, local)

## v2.0 — Platform (Q4 2026)

- [ ] Team workspaces and SSO
- [ ] SaaS hosted offering
- [ ] Usage-based billing
- [ ] API for embedding OpsAI in other tools
- [ ] Mobile app for on-call engineers
- [ ] Reliability scoring per repository

## Monetization Strategy

| Tier | Price | Features |
|------|-------|----------|
| Free | $0 | 3 projects, 50 runs/mo, community support |
| Pro | $29/mo | Unlimited projects, 1000 runs/mo, Slack + email |
| Team | $99/mo | SSO, team workspaces, priority support |
| Enterprise | Custom | On-premise, SLA, custom integrations |
